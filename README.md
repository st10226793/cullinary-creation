# Culinary Creations - Expo (SDK 54)

## Project Overview
Culinary Creations is a mobile application designed to inspire users in their cooking journey. The app allows users to track their culinary creations, stay motivated with daily quotes, and manage their cooking goals. With intuitive navigation and a user-friendly interface, Culinary Creations enhances the cooking experience.

## Features
- **Bottom Tabs Navigation**: Seamless navigation through main sections including Home, Menu Progress, Orders, Profile, and Settings.
- **Menu Item Tracker**: Track the number of menu items created; automatically saved using AsyncStorage.
- **Daily Culinary Inspiration Quotes**: Receive a new motivational quote each day to inspire your cooking.
- **Goal Tracking for Menu Items**: Set and persist weekly cooking goals to keep you motivated.
- **Profile Screen**: Save user information, including name and cooking goals.
- **Dark Mode Toggle**: Easily switch between light and dark themes; preferences are saved.
- **Well-Commented Code**: All code includes inline comments to facilitate learning and understanding.

## Technologies Used
- **React Native**: Framework for building native apps using React.
- **Expo**: Toolchain for developing and building React Native apps.
- **AsyncStorage**: For persisting user data locally on the device.
- **React Navigation**: For managing navigation within the app.
- **React Hooks**: For managing state and lifecycle features in functional components.
